# dnd-charsheet

### client

Install elm globally

```
npm install -g elm
```

Install the dnd elm packages
```
elm install
```

Run the app
```
elm reactor
```


Build the javascript
```
elm make src/Main.elm --output public/dnd.js
```

### server

Standard server
```
npm run start
```
Dev Server (auto restart)
```
npm run start-dev
```
